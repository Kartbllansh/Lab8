M    help : вывести справку по доступным командам
M    info : вывести в стандартный поток вывода информацию о коллекции (тип, дата инициализации, количество элементов и т.д.)
M    show : вывести в стандартный поток вывода все элементы коллекции в строковом представленииM   add {element} : добавить новый элемент в коллекцию
М     update id {element} : обновить значение элемента коллекции, id которого равен заданному
M    remove_by_id id : удалить элемент из коллекции по его id
M    clear : очистить коллекцию
M     save : сохранить коллекцию в файл
     execute_script file_name : считать и исполнить скрипт из указанного файла. В скрипте содержатся команды в таком же виде, в котором их вводит пользователь в интерактивном режиме.
M    exit : завершить программу (без сохранения в файл)
M     remove_at index : удалить элемент, находящийся в заданной позиции коллекции (index)
M    add_if_min {element} : добавить новый элемент в коллекцию, если его значение меньше, чем у наименьшего элемента этой коллекции
     remove_greater {element} : удалить из коллекции все элементы, превышающие заданный
P    remove_any_by_type type : удалить из коллекции один элемент, значение поля type которого эквивалентно заданному
M    print_field_ascending_age : вывести значения поля age всех элементов в порядке возрастания
M    print_field_descending_type : вывести значения поля type всех элементов в порядке убывания
